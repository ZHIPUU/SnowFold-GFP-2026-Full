## 六、对项目设计的洞察

### 1. **多轮 MPNN 的收敛路径**

R14 → R19 → R20 看到同样趋势：序列越来越偏离 sfGFP/avGFP WT，但保持 β-barrel 核心结构。

### 2. **优化目标不止"恢复亮度"**

最终 R20 候选平均 score = 0.939，但与 WT 的 identity 仅 ~37% — 这是**全新设计的 GFP**，而不是 sfGFP 突变体。

### 3. **chromophore 5-core 不是固定的**

虽然 fixed positions [1, 65, 66, 67, 96, 222] 是固定的，但 R20 父代的 pos 65 已经是 L（不是 T），这说明前几轮 MPNN 把这一位置从 T 收敛到了 L。

### 4. **可重现的优化路径**

10 个保守突变（S3I, K4P, F9L 等）几乎所有 1100+ 候选都共享 — 这是一个**稳健的最小突变集**，可以手动应用做"快速变体设计"。
