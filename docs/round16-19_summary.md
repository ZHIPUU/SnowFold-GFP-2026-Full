## 三、关键技术发现汇总

### 3.1 pLDDT 评分尺度 (R17 发现)

| 实现 | 输出 | 公式 |
|:-----|:-----|:----|
| fair-esm | 0-100 整数 | `0.3 * pLDDT/100` |
| HuggingFace | 0-1 浮点 | `0.3 * pLDDT` |

**教训**: R15 "综合分 0.983" 实际是 bug,真实 0.892。

### 3.2 ProteinMPNN fixed_positions key (R18 发现)

```python
# 错误: 包含全路径
pdb_key = pdb_path.replace(".pdb", "")

# 正确: 必须用 basename
pdb_key = os.path.basename(pdb_path).replace(".pdb", "")
```
