## 五、关键洞察

1. **"近原点搜索"是最高效策略**: 从 R10 父代出发，仅 28 突变就达到 0.881
2. **温度 T=0.3 是甜蜜点**: 平衡保守与多样性
3. **跨骨架多样性需要先"改造骨架"**: avGFP WT 不能直接跑 MPNN
4. **C 端尾仍是 pLDDT 天花板**: 即使 R11 Top 1 整体 84.2，C 端尾仍 50-60
5. **过拟合风险显著**: 6 条候选全部来自同一父代，亟需多样性补救
